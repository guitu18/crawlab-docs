修改节点信息

后面我们需要让爬虫运行在各个节点上，需要让主机与节点进行通信，因此需要知道节点的IP地址和端口。我们需要手动配置一下节点的IP和端口。在节点列表中点击操作列里的蓝色查看按钮进入到节点详情。节点详情样子如下。

在右侧分别输入该节点对应的节点IP和节点端口，然后点击保存按钮，保存该节点信息。

这样，我们就完成了节点的配置工作。
